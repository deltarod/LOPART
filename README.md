## Installing
This section will explain how to install pyLOPART

### With Pip

`python3 -m pip install git+https://github.com/deltarod/LOPART.git`

### From Sources

1. `git clone https://github.com/deltarod/LOPART.git`
2. `cd LOPART/`
3. `python3 -m pip install .`
